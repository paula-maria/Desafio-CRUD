# CRUD de Produtos 
Este projeto foi desenvolvido como parte do **Desafio CRUD**, implementando um sistema completo de **cadastro, listagem, atualização, exclusão e pesquisa** de produtos desenvolvido com **Node.js**, **Express** e **SQLite**. 

## Tecnologias
- Node.js
- Express
- SQLite
- HTML, CSS e JavaScript (Fetch API)

## Funcionalidades
- Listar todos os produtos  
- Exibir um produto específico  
- Criar, atualizar e excluir produtos  
- Pesquisa de produtos por nome  
- Validação de dados (nome obrigatório, preço e quantidade positivos
